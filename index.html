Download-Kategorie hinzufügen

Kategorie erstellen

Im Backend auf die Übersichtsseite der Kategorien gehen, indem man mit der Maus über Beiträge geht und im erscheinenden Menü auf Kategorien klickt.

Hier sieht man eine Liste aller Kategorien, die auf der Website angelegt wurden. In der Liste sind Unterkategorien mit einem Minus davor dargestellt. Im nächsten Screenshot sieht man die Downloadkategorien-Kategorie samt ihren Unterkategorien.

Hier soll jetzt eine neue Unterkategorie erstellt werden indem man das Formular auf der linken Hälfte des Bildschirms verwendet. In diesem Fall wurde eine Kategorie namens Anmeldescheine Übergangsklasse erstellt. Hier ist es wichtig die Übergeordnete Kategorie Downloadkategorien auszuwählen, damit die Kategorien auch in Zukunft übersichtlich bleiben.

Indem man das Formular mit dem Neue Kategorie erstellen-Button absickt, wird die neue Kategorie erstellt und die Kategorie wird jetzt erscheinen, wenn man einen neuen Download anlegt oder einen existierenden bearbeitet.

Somit kann man jetzt die Downloads anlegen, die dann in der Kategorie erscheinen sollen.

Änderungen auf der Download-Seite

Bei Änderungen auf der Download-Seite sollte immer eine bestehende Komponente dupliziert werden und dann die entsprechenden Änderungen auf der duplizierten Variante angewendet werden. Damit wird sichergestellt, dass alle Komponenten gleich aussehen und die Komponenten sich nur durch die sich auszeichnenden Einstellungen unterscheiden.

Man kann Komponenten duplizieren indem man einen Rechtsklick auf die Komponente macht und im erscheinenden Kontext-Menü Duplizieren auswählt. Das neue Element wird direkt unter dem ausgewählen Element eingefügt und kann nun an den Platz geschoben werden, wo man es haben möchte.

In unserem Fall möchten wir die Überschrift “Unterstufe” duplizieren und in “Übergangsklasse” ändern. Dabei soll die Überschrift unter die Liste von Unterstufen-Downloads geschoben werden.

Jetzt kann man die Liste an Unterstufen-Downloads duplizieren und unter die erstellte Überschrift ziehen. Der letzte Schritt ist die Bearbeitung der Einstellungen dieser duplizierten Liste an Downloads. Mit einem Klick auf den Bleistift im rechten oberen Eck der Komponente erreicht man die Einstellungen, welche in der Sidebar links dargestellt werden.

Jetzt muss nur noch die richtige Downloadkategorie ausgewählt werden. Diese Einstellung findet man unter dem Tab “Inhalt” im Abschnitt “Abfrage”. Dort kann man im Feld Term nach dem Namen der neu angelegten Kategorie suchen. Mit einem Klick wählt man diese aus und hat damit die Einstellung vorgenommen.

Jetzt muss man nur noch die Seite speichern und die Kategorie erscheint auf der produktiven Seite.
